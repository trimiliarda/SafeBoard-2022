- Используем make instull, и в открывшемся vim'е записываем такой скрипт:

"""

alias finder() {
  gcc finder.c -o ~/find;
  ~/./find "$1";
  rm ~/find;
}

"""

- Дальше используем ". ~/.bash_profile" и после этого пользуемся командой и программой. 
- Всё готово, можно использовать команду.
